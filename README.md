# Personal Expense Tracker

## Problem Statement
Many students and young professionals struggle to track daily expenses, 
leading to poor financial awareness and overspending. This tool provides 
a simple way to record, categorize, and analyze spending patterns.

## Features
- Add expenses with amount, description, category, and date
- View all expenses in chronological order
- Delete individual expenses
- Filter expenses by category
- Real-time summary showing total spending and top category
- Data persists across browser sessions using LocalStorage
- Input validation and error handling

## Technologies Used
- HTML5
- CSS3 (Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- LocalStorage API

## How to Run
Go To https://sanidhyapadhy.github.io/expense_tracker_end_term_project/ & Start adding expenses!
